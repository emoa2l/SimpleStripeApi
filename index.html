SimpleStripeApi

Stripe payment processing library for .NET 8+

GitHub | Documentation | License

Installation

dotnet add reference path/to/SimpleStripeApi/SimpleStripeApi.csproj

Usage

builder.Services.AddStripeTransactionService();
app.MapStripeTransactionEndpoints();

Features

  • Single endpoint for payment processing
  • Async transaction handling
  • JSON request/response
  • Built on Stripe.net SDK
  • Minimal configuration required

Requirements

  • .NET 8.0+
  • Stripe.net v45.9.0+
  • Stripe API key

Example

POST /api/stripe/transaction
{
  "amount": 2500,
  "currency": "usd",
  "paymentMethodId": "pm_card_visa"
}
